- Knowledge areas remain; application is shifting in a market-driven economy.
- HR roles vs framework roles (Product Owner etc.); someone has to step into framework roles.
- Core values (PMI code of ethics: respect, fairness, responsibility, trust, transparency).
- Core vs functional competencies; references to IAEA and Australian DoD competency standards.
